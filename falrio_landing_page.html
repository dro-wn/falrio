Privacy by default. Blocking by design.

A Chromium-based browser with real tracker blocking, AI assistance, and genuine privacy controls. No corporate tracking. No compromises.

🛡️ Real Tracker Blocking

~60 known tracker/ad domains blocked at the network level via declarativeNetRequest. No fake counters—real protection.

🔐 Fingerprint Protection

Canvas and audio fingerprinting noise injection breaks common tracking scripts without breaking normal browser use.

🤖 AI Sidebar

Summarize pages, explain selections, or ask questions without leaving the tab. Works on every website.

🔄 HTTPS Upgrade

All http:// requests are automatically upgraded to https:// for safer browsing.

📊 Live Privacy Stats

See real counts of trackers and cookies blocked. No simulation, no fake numbers.

🎨 Custom New Tab

Beautiful, functional New Tab page with weather, calendar, notes, pomodoro timer, and more.

Installation

1

Download Chromium or Chrome

If you don't have Chrome/Edge/Brave installed, download Chrome here.

2

Download Falrio Extension

Go to the GitHub repo and click the green Code button → Download ZIP.

Extract the ZIP file. You'll see a folder called falrio-extension-main.

3

Open Extensions Page

In Chrome/Edge, go to chrome://extensions

4

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner.

5

Load the Extension

Click Load unpacked and select the extension folder inside falrio-extension-main.

6

Done!

Open a new tab. You should see the Falrio New Tab page. Click the gold circle on any website to open the AI sidebar.
