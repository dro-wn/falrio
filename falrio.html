What it does

Built to actually protect you

🛡️

Real tracker blocking

~60 known tracker/ad domains blocked at the network level — enforced by Chromium itself, not a JS simulation.

🔐

Fingerprint noise

Canvas & audio readback get imperceptible noise, breaking common fingerprinting scripts without breaking normal use.

🔄

Forced HTTPS

Every plain http:// request gets automatically upgraded to https:// before it ever loads.

🤖

AI sidebar, everywhere

Summarize a page, explain a selection, or ask a question — on any website, one click away.

📊

Live privacy dashboard

Real counts of what got blocked this session. Numbers you can trust, not for-show placeholders.

🎨

A New Tab worth keeping

Weather, notes, a pomodoro timer, world clock, and shortcuts — fast, clean, and yours to customize.

Get started

Install in under 2 minutes

1

Get a Chromium-based browser

Chrome, Edge, or Brave all work. Download Chrome if needed.

2

Download Falrio

On the GitHub repo, click Code → Download ZIP, then extract it.

3

Open extensions page

Go to chrome://extensions

4

Turn on Developer mode

Flip the switch, top-right corner.

5

Load unpacked

Click it, then select the extension folder from the extracted ZIP.

6

Open a new tab

That's Falrio. Click the gold circle on any site for the AI sidebar.
